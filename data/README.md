# Data placeholder

The original project uses external county-level natality and demographic data. The full raw files are not committed to this repository because of size.

Required files:
- `2011.txt.gz` to `2020.txt.gz` from CDC WONDER natality exports grouped by county
- `us.1990_2023.20ages.adjusted.txt` from SEER county population files
- `ruralurbancodes2013.xls`
- `US_2022_ADI_Census_Block_Group_v4_0_1.csv.gz`

See the repository README for download notes.
